# Piranha

Apple Wallet pass generator with a nice interface.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     piranha:
       github: grkek/piranha
   ```

2. Run `shards install`

## Usage

```crystal
require "piranha"
```
